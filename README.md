# 🥟 딤섬 클리커 (Dimsum Clicker)

딤섬 찜기 모양의 3단 적층형 하드웨어 케이스. ESP32-C3 + 520mAh 배터리 + OLED + 키보드 스위치를 담아
누르면 반응하는 미니 가젯을 만드는 프로젝트입니다.

**브라우저에서 슬라이더로 케이스를 실시간 재설계**하고 층별 STL을 바로 뽑을 수 있는 웹 컨피규레이터가 핵심입니다.

```
1층 (배터리)  ─  2층 (ESP32 · 충전모듈 · OLED · USB)  ─  3층 (뚜껑 + 스위치 스탠드 + 딤섬 캐릭터)
```

---

## 실행 방법

브라우저(three.js + WASM) 앱이라 **로컬 웹 서버만 있으면** 됩니다. 별도 설치·빌드가 없습니다.

```bash
# 1) 프로젝트 루트에서 실행 (stl_files/ 를 읽어야 하므로 반드시 루트에서)
cd ESP-with-3dprinter
python3 -m http.server 8123

# 2) 브라우저에서 접속
#    http://localhost:8123/web/
```

> Python이 없으면 아무 정적 서버나 가능합니다 — 예: `npx serve` 후 `/web/` 경로로 접속.
> 필요한 라이브러리(three.js·manifold WASM 등)는 전부 `web/vendor/`에 포함되어 있어 **인터넷 없이도** 동작합니다.

### 쓰는 순서

1. 왼쪽 패널에서 모양(둥근 네모/원형)·크기·층 높이·부품 위치를 조정 → 3D 미리보기가 실시간 갱신
2. `배선 표시`를 켜면 배터리→충전모듈→ESP32→OLED→스위치 배선이 색깔 전선으로 표시됨
   (보라 스위치 · 초록 SDA · 주황 SCL 전선을 **우클릭**하면 GPIO 핀 변경)
3. `분해 ⟷ 조립` 슬라이더로 조립 상태 확인, 하단의 `조립 간섭 없음 ✓` 확인
4. `STL 내보내기`에서 1·2·3층을 각각 다운로드 → 슬라이서로 출력
5. 마음에 드는 설정은 `프리셋 → 내보내기`로 .json 저장 (설정은 자동으로도 저장됨)

전체 기능 설명은 [`web/README.md`](web/README.md) 참고.

## 프로젝트 구조

| 경로 | 내용 |
|---|---|
| `web/` | **웹 컨피규레이터** (메인). `index.html` + `app.js` + `vendor/`(로컬 라이브러리) |
| `generate_dimsum_clicker.py` | 파이썬 STL 생성기 (웹과 동일한 manifold 엔진, 배치 출력용) |
| `stl_files/` | 부품 실측 STL (배터리·충전모듈·ESP32·스위치 스탠드·OLED·USB 구멍) — 케이스 절삭/배치 기준 |
| `hardwares/` | 부품 데이터시트/사진 (핀아웃, 치수) |
| `dimsum/` | 딤섬 캐릭터·클릭 캡 STL |
| `my_designs/` | 초기 손 모델링 참고본 (원형 다마고치 스타일 등) |

## 두 가지 생성 방식

- **웹 컨피규레이터** (`web/`) — 실시간 파라메트릭. 슬라이더로 조정하고 즉시 STL 다운로드. 평소엔 이걸 쓰세요.
- **파이썬 생성기** (`generate_dimsum_clicker.py`) — 고정 파라미터로 배치 생성.
  ```bash
  pip3 install --user trimesh manifold3d shapely
  python3 generate_dimsum_clicker.py   # output_stl/ 에 층별 STL 생성
  ```

둘 다 **manifold WASM/라이브러리**로 CSG 연산을 해서 출력 STL이 항상 watertight(출력 가능)입니다.

## 필요한 하드웨어 (BOM)

| 부품 | 사양 | 케이스 반영 |
|---|---|---|
| ESP32-C3 SuperMini | 24 × 18 × 4.2mm | 2층 포켓 + 핀아웃 기반 배선 |
| 리튬 배터리 | 520mAh, 36.5 × 28.5 × 4.3mm | 1층 포켓 (대각선 46.3mm → 원형 최소 Ø54) |
| USB-C 충전/보호 모듈 | 19 × 14 × 4.5mm | 2층, USB 포트가 벽 구멍/플랫 패드로 노출 |
| OLED 디스플레이 | 0.49" (15×16) 또는 0.96" (26×26) | 2층 타워에 뒤에서 삽입, 창으로 화면 노출 |
| 키보드 스위치 + 스탠드 | 17.79 × 14.3, 높이 11mm | 3층 뚜껑에 실물 형상대로 구멍 |

배선(웹의 `배선 표시` 참고):
- 배터리 +/− → 충전모듈 **B+/B−**
- 충전모듈 **OUT+/OUT−** → ESP32 **5V/GND**
- ESP32 **3V3·GND·GPIO8·GPIO9** → OLED **VCC·GND·SDA·SCL** (I2C, 핀 변경 가능)
- 스위치 2가닥 → ESP32 **GPIO4 + GND** (극성 무관, 핀 변경 가능)

```cpp
// 펌웨어 예시
Wire.begin(8, 9);          // SDA, SCL — 웹에서 바꾼 핀 그대로
pinMode(4, INPUT_PULLUP);  // 스위치: 눌리면 LOW
```

## 출력 & 조립 팁

- **1·2층은 그대로**, **3층만 뒤집어서** 출력 (상판이 베드에 닿게 — 서포트 불필요). 내보내기의 `3층 출력용 뒤집기` 체크가 처리해 줌.
- **V형 결합부**(기본 켜짐)를 쓰면 2층 밑면 결합 홈에 **서포트가 필요 없고** 층이 얹을 때 자가 정렬됩니다.
- 층이 **헐거우면** `결합 유격`을 낮추고(기본 0.08mm), **안 들어가면** 올리세요. 프린터 수축에 맞춰 조정.
- 각 층은 딤섬 찜기처럼 **얹으면 맞물리는** 구조라 나사·접착 없이 조립됩니다.
