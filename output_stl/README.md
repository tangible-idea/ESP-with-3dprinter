# 딤섬 찜기 클리커 — 생성된 STL

`python3 generate_dimsum_clicker.py` 실행으로 재생성 가능. 치수 수정은 스크립트 상단 파라미터 참고.

## 파일

| 파일 | 내용 | 크기 |
|---|---|---|
| `floor1_battery.stl` | 1층: 520mAh 배터리 수납 | Ø54 × 11.5mm |
| `floor2_esp32.stl` | 2층: ESP32 + 충전모듈 + OLED 소켓 + USB-C 구멍 | Ø54 × 23.5mm |
| `floor3_switch_lid.stl` | 3층: 스위치 스탠드 장착 뚜껑 | Ø54 × 12mm |
| `assembly_preview.stl` | 조립 확인용 (출력 금지) | — |

## 구조

- **외경 54mm / 벽 2.5mm / 내경 49mm** — 배터리(36.5×28.5) 대각선 46.3mm가 들어가도록 기존 41mm 디자인에서 키움
- **찜기식 스택 결합**: 각 층 상단에 결합 링(lip, 1.5mm), 위층 바닥에 홈(groove) — 그냥 얹으면 맞물림. 옆면엔 대나무 찜기 느낌의 가로 홈 2줄
- **1층**: 배터리 포켓(37×29, 여유 0.5). 천장(=2층 바닥판)의 배선 구멍으로 선이 2층으로 올라감
- **2층**:
  - 바닥판에 배선 구멍 9×5 (남쪽, y=-13.5) → 1층 배터리 선 통과
  - ESP32 포켓 24.4×18.4 (서쪽), 충전모듈 포켓 19.4×14.4 (동쪽, USB가 벽 방향)
  - 동쪽 벽: `usb_c_hole.stl` 형상으로 USB-C 관통 구멍 (충전모듈 포트 정면)
  - 북쪽 벽: OLED(15×16×2.4) 세로 삽입 슬롯 — 위에서 밀어넣기, 아래에 핀 배선 통로, 외벽에 13.5×8 디스플레이 창
- **3층 (뚜껑)**: 상판(두께 3.2)에 `keyboard_switch_stand.stl` 바닥 결합부 그대로 구멍
  - 중앙 7×7 관통(기둥 5.3×5.3 + 배선 통로), 다리 구멍 2.3×2.3 ×2, 안착 홈 14.9×15.3×0.6
  - 스위치 배선은 중앙 구멍으로 2층에 떨어짐

## 출력 팁

- 1층, 2층: 그대로 출력 (바닥판이 아래)
- **3층은 뒤집어서 출력** (상판이 베드에 닿게) — 안 뒤집으면 상판이 공중 브릿지가 됨
- 바닥 결합 홈(폭 1.4mm)은 짧은 브릿지라 서포트 불필요
- 공차: 포켓 0.4, 스택 결합 0.2/측 — 프린터에 따라 스크립트의 `CLR`, `LIP/GRV` 값 조정
